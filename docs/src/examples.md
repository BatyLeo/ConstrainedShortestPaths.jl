# Examples

## Usual shortest path
- ``q_o^F = 0,\, q^B_d = 0``
- ``f^F_a(q) = f^B_a(q) = c_a + q``
- ``c(q^F, q^B) = q^F + q^B``

## Usual resource constrained shortest path
- ``q_o^F = 0,\, q^B_d = 0``
- ``f^F_a(q) = f^B_a(q) = c_a + q``
- ``c(q^F, q^B) = q^F + q^B + \mathbb{I}_{\{q^F + q^B \leq W\}}``
